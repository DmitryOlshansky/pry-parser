auto  buffer()(ubyte[] data);
auto  buffer(T)(T[] data) if (is(Unqual!T == ubyte));
auto  buffer(T)(T[] data) if (is(Unqual!T == char));

Wrap an array as buffer. If the original array had immutable elements then the resulting buffer supports zero-copy slicing.

See also dpick.buffer.traits.isBuffer and dpick.buffer.traits.NullBuffer.

auto  buffer(Input)(Input stream, size_t minHistory = 32, size_t bufferSize = 8 * 1024, size_t page = 512) if (isInputStream!Input);

Creates a bufer range that takes ownership of stream input stream. Tweakable parameters include initial buffer size, a block size of the buffer, and minimal history (in bytes) to keep for lookbehind during buffering.

See also dpick.buffer.stream.fileInput, dpick.buffer.traits.isInputStream, dpick.buffer.traits.isBuffer, and dpick.buffer.traits.NullBuffer.
